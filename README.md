# Family Shield 🛡️
### Router-Based Parental Control System

A graduation project that provides parents with a simple web dashboard to manage and monitor their home network — no technical knowledge required.

---

## Overview

Family Shield runs on an OpenWrt router and gives parents full control over connected devices through a clean, easy-to-use interface. Controls apply at the network level, covering all Wi-Fi and Ethernet devices from one place.

---

## Features

- 📋 **Device Management** — Detect connected devices and assign readable names
- ⏸️ **Pause Internet** — Block or resume any device instantly
- 🕒 **Scheduling** — Set daily internet time windows per device
- 🔒 **Content Filtering** — DNS-based category filtering (dnsmasq)

---

## Tech Stack

| Layer | Technology |
|---|---|
| Router OS | OpenWrt |
| Backend | Python, Flask |
| Frontend | HTML, CSS, JavaScript |
| Storage | JSON (local) |
| Router Services | dnsmasq, iptables, cron, DHCP, ARP |
